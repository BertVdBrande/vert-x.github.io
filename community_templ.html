Project and Community Information

The Project

The Vert.x core project is a project of the Eclipse Foundation The Eclipse project proposal provides some useful background information including a list of interested companies and individuals.

The core project committers include developers employed by Red Hat and Pivotal Inc., and committers to other Vert.x projects include developers from several other companies.

There is more information on the Wikipedia page

Vert.x has a thriving community and includes further contributions by many other individuals and organisations

The Source

The Vert.x source lives at github where the vert-x organisation is the umbrella under which all the Vert.x related projects live. This includes the core Vert.x project, as well as various others such as language implementation modules, the test framework, web-site and others.

Vert.x is highly modular and the core project is quite compact. Much of the functionality of Vert.x is provided in the form of modules - some maintained under the vert-x umbrella but many others created and maintained by the community.
